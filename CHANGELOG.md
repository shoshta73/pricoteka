# Changelog

## 0.1.0

- Added the first Pricoteka app release for tracking stores, offices, and product prices.
- Added a dashboard with store, office, and product totals plus quick actions.
- Added store and product list views with empty states and create actions.
- Added an About page with app information, storage mode details, version display, and license text.
- Added app-wide dark and light theme support.
- Added app navigation improvements with breadcrumbs, a sidebar layout, and a compact footer.
- Added the first public Pricoteka website with a concise product overview.
- Added dark and light theme support on the website.
- Added a cleaner website header, preview card, and footer.
